## 5 Content Domains Overview

The AIP-C01 exam consists of 5 Content Domains with the following scoring weightings:

| Domain | Name | Weighting |
|--------|------|-----------|
| Domain 1 | Foundation Model Integration, Data Management, and Compliance | **31%** |
| Domain 2 | Implementation and Integration | **26%** |
| Domain 3 | AI Safety, Security, and Governance | **20%** |
| Domain 4 | Operational Efficiency and Optimization for GenAI Applications | **12%** |
| Domain 5 | Testing, Validation, and Troubleshooting | **11%** |

{{% notice tip %}}
**Exam Strategy:** Domain 1 (31%) and Domain 2 (26%) account for **57%** of the total score. Prioritize studying these two domains first. Domain 3 (20%) is also critical as it covers security â€” a topic AWS always emphasizes.
{{% /notice %}}

### Domain 1: Foundation Model Integration, Data Management, and Compliance (31%)

The largest domain, focusing on:
- Analyzing requirements and designing GenAI solutions
- Selecting and configuring Foundation Models
- Building data validation and processing pipelines
- Designing vector store solutions
- Designing retrieval mechanisms (RAG)
- Prompt engineering strategies and governance

**Key AWS Services:** Amazon Bedrock, Amazon Bedrock Knowledge Bases, Amazon OpenSearch Service, Amazon SageMaker AI, AWS Glue, AWS Lambda

### Domain 2: Implementation and Integration (26%)

Focuses on practical implementation:
- Agentic AI solutions and tool integrations
- Model deployment strategies
- Enterprise integration architectures
- FM API integrations
- Application integration patterns

**Key AWS Services:** Amazon Bedrock, AWS Step Functions, AWS Lambda, Amazon API Gateway, Amazon ECS, AWS CodePipeline

### Domain 3: AI Safety, Security, and Governance (20%)

AI security and governance:
- Input/output safety controls
- Data security and privacy controls
- AI governance and compliance
- Responsible AI principles

**Key AWS Services:** Amazon Bedrock Guardrails, Amazon Comprehend, Amazon Macie, IAM, AWS CloudTrail, Amazon CloudWatch

### Domain 4: Operational Efficiency and Optimization (12%)

Operational optimization:
- Cost optimization and resource efficiency
- Application performance optimization
- Monitoring systems for GenAI

**Key AWS Services:** Amazon CloudWatch, AWS Cost Explorer, Amazon Bedrock (provisioned throughput), AWS Auto Scaling

### Domain 5: Testing, Validation, and Troubleshooting (11%)

Testing and troubleshooting:
- Evaluation systems for GenAI
- Troubleshooting GenAI applications

**Key AWS Services:** Amazon Bedrock Model Evaluations, Amazon CloudWatch Logs, AWS X-Ray, Amazon SageMaker Clarify
