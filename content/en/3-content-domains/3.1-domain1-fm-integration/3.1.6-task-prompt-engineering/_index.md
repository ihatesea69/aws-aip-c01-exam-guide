## Task 1.6: Prompt Engineering Strategies and Governance

### Skill 1.6.1: Model Instruction Frameworks

**Key Knowledge:**
- **Amazon Bedrock Prompt Management** — Enforce role definitions, parameterized templates
- **Amazon Bedrock Guardrails** — Enforce responsible AI guidelines
- System prompts, user prompts, assistant prompts

### Skill 1.6.2: Interactive AI Systems

**Key Knowledge:**
- **Step Functions** for clarification workflows
- **Amazon Comprehend** for intent recognition
- **DynamoDB** for conversation history storage

### Skill 1.6.3: Prompt Management and Governance

**Key Knowledge:**
- **Amazon Bedrock Prompt Management** — Parameterized templates, approval workflows
- **AWS CloudTrail** — Track prompt usage

{{% notice info %}}
**Prompt Management** in Bedrock allows creating, versioning, and managing prompts centrally. This is a best practice for enterprise deployments and important exam knowledge.
{{% /notice %}}

### Skill 1.6.4: Quality Assurance for Prompts

**Key Knowledge:**
- Lambda functions to verify expected output
- A/B testing for prompt variants

### Skill 1.6.5: Enhance FM Performance Through Prompt Refinement

**Key Knowledge:**
- Structured input components
- Output format specifications (JSON mode, XML tags)
- **Chain-of-thought** instruction patterns
- Few-shot prompting

### Skill 1.6.6: Complex Prompt Systems

**Key Knowledge:**
- **Amazon Bedrock Prompt Flows** — Sequential prompt chains
- Conditional branching based on model responses
- Reusable prompt components
