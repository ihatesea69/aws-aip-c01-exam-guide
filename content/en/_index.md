# AWS Certified Generative AI Developer - Professional (AIP-C01)

### Introduction

A comprehensive workshop to prepare for the AWS Certified Generative AI Developer - Professional (AIP-C01) exam. This workshop covers detailed analysis of all 5 Content Domains, hands-on labs with Amazon Bedrock and AWS GenAI services, reference materials, and exam strategies.

### Requirements

- AWS Account with Administrator access
- Basic AWS knowledge (compute, storage, networking)
- 2+ years experience building applications on AWS
- 1+ year experience implementing GenAI solutions
