## Create VPC

Steps to create a VPC for the workshop.

1. Go to [VPC Console](https://console.aws.amazon.com/vpc/)
2. Click **Create VPC**
3. Configure:
   - Name: `workshop-vpc`
   - IPv4 CIDR: `10.0.0.0/16`

![Create VPC](/images/sample/create-vpc.png)

{{% notice tip %}}
You can use the VPC wizard for a quicker setup.
{{% /notice %}}
