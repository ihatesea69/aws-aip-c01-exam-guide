## Create EC2 Instance

Steps to create an EC2 instance.

1. Go to [EC2 Console](https://console.aws.amazon.com/ec2/)
2. Click **Launch Instance**
3. Configure according to workshop requirements

![Create EC2](/images/sample/create-ec2.png)

{{% notice warning %}}
Remember to choose a Free Tier eligible instance type to avoid unexpected charges.
{{% /notice %}}
