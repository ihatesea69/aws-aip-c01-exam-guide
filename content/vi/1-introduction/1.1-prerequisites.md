## Yêu cầu trước khi bắt đầu

### AWS Account

- Bạn cần một AWS Account. Nếu chưa có, [tạo tại đây](https://aws.amazon.com/free/).
- Sử dụng IAM user với quyền Administrator (không dùng root account).

### Kiến thức cần có

- Hiểu biết cơ bản về AWS Console
- Kiến thức cơ bản về networking (VPC, subnet)

### Công cụ

| Tool | Mô tả |
|------|-------|
| AWS CLI | Command line interface |
| Browser | Chrome/Firefox recommended |
