## Tạo VPC

Mô tả các bước tạo VPC cho workshop.

1. Truy cập [VPC Console](https://console.aws.amazon.com/vpc/)
2. Chọn **Create VPC**
3. Cấu hình:
   - Name: `workshop-vpc`
   - IPv4 CIDR: `10.0.0.0/16`

![Create VPC](/images/sample/create-vpc.png)

{{% notice tip %}}
Bạn có thể sử dụng VPC wizard để tạo nhanh hơn.
{{% /notice %}}
