## Tạo EC2 Instance

Mô tả các bước tạo EC2 instance.

1. Truy cập [EC2 Console](https://console.aws.amazon.com/ec2/)
2. Chọn **Launch Instance**
3. Cấu hình theo yêu cầu workshop

![Create EC2](/images/sample/create-ec2.png)

{{% notice warning %}}
Nhớ chọn instance type phù hợp với Free Tier nếu bạn muốn tránh phát sinh chi phí.
{{% /notice %}}
