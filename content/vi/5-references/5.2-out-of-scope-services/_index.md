## AWS Services Out-of-Scope cho kỳ thi AIP-C01

Các services sau **không** nằm trong phạm vi kỳ thi. Không cần ôn các services này.

{{% notice info %}}
Danh sách này không đầy đủ. Các AWS services hoàn toàn không liên quan đến vai trò GenAI developer đã được loại trừ.
{{% /notice %}}

### Application Integration
- Amazon MQ

### Analytics
- AWS Clean Rooms, AWS Data Exchange, Amazon DataZone, Amazon FinSpace

### Blockchain
- Amazon Managed Blockchain (AMB)

### Business Applications
- Alexa for Business, Amazon Chime, AWS Wickr, Amazon WorkDocs, Amazon WorkMail

### Cloud Financial Management
- AWS Budgets, AWS Cost and Usage Report, Reserved Instance reports, AWS Savings Plans

### Compute
- AWS Batch, Amazon EC2 Image Builder, Amazon ECS Anywhere, Amazon EKS Anywhere, AWS Elastic Beanstalk, Amazon Lightsail, AWS Local Zones, AWS Serverless Application Repository

### Containers
- AWS App2Container, AWS Copilot, Red Hat OpenShift Service on AWS (ROSA)

### Database
- Amazon Keyspaces, Amazon QLDB, Amazon Redshift, Amazon Timestream

### Developer Tools
- AWS Cloud9, AWS CloudShell, Amazon CodeGuru, AWS CodeStar, Amazon Corretto

### Machine Learning (Out-of-scope)
- AWS DeepComposer, AWS DeepRacer, Amazon DevOps Guru, Amazon Forecast, Amazon Fraud Detector, Amazon HealthLake, Amazon Lookout (Equipment/Metrics/Vision), Amazon Monitron, AWS Panorama

### IoT
- Tất cả AWS IoT services (Core, Greengrass, SiteWise, TwinMaker, etc.)

### Media Services
- Tất cả AWS Elemental services, Amazon Kinesis Video Streams

### Migration and Transfer
- AWS Application Discovery/Migration Service, CloudEndure, AWS Migration Hub, AWS Snow Family

### Networking
- AWS App Mesh, AWS Cloud Map, AWS Direct Connect, AWS Private 5G, AWS Transit Gateway, AWS VPN

### Khác
- Amazon Braket (Quantum), AWS RoboMaker (Robotics), AWS Ground Station (Satellite)
